# 📊 Superstore Sales Dashboard (Excel)
Interactive Excel dashboard analyzing Sales over years

## 📊 Overview
This project presents an interactive Excel dashboard analyzing sales performance, profitability, product categories, and regional trends using the Superstore dataset. The dashboard provides clear business insights to support data-driven decision-making.

## 🔍 Key Insights
- High sales but low profit observed in certain sub-categories  
- Technology category contributes the highest profit  
- Profitability varies significantly across regions  
- Sales show noticeable seasonal fluctuations  
- Higher discounts negatively impact profit margins  

## 💡 Recommendations
- Optimize pricing and reduce excessive discounts  
- Focus on high-margin products in the Technology category  
- Improve operational efficiency in low-profit regions  
- Align inventory and promotions with seasonal demand  
- Implement controlled discount strategies  

## 🛠 Tools Used
- Microsoft Excel
